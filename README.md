# ☁️ Azure Virtualization Project

## 📖 Descripción

Proyecto enfocado en la implementación de infraestructura en la nube mediante máquinas virtuales en Microsoft Azure.

Se desplegaron múltiples servicios configurados manualmente dentro de entornos virtualizados.

---

## 🚀 Arquitectura del proyecto

El sistema está compuesto por:

* 🌐 **Servidor Web (NGINX)**
* 🗄️ **Servidor de Base de Datos (MySQL)**
* 📂 **Servidor FTP (FileZilla)**

Cada servicio fue configurado en una máquina virtual independiente dentro de Azure.

---

## 🛠️ Tecnologías utilizadas

* Microsoft Azure
* Máquinas Virtuales (VM)
* NGINX
* MySQL
* FileZilla (FTP)
* Linux (Ubuntu)

---

## ⚙️ Implementación

* Creación y configuración de máquinas virtuales en Azure
* Instalación de servidor web con NGINX
* Configuración de base de datos MySQL
* Implementación de servidor FTP para transferencia de archivos
* Conexión y validación de servicios

---

## 📚 Aprendizaje

* Despliegue de infraestructura en la nube

* Manejo de entornos virtualizados en Azure

## 🎯 Conclusión

Este proyecto permitió comprender el funcionamiento de una arquitectura básica en la nube, integrando múltiples servicios en máquinas virtuales dentro de Azure.
